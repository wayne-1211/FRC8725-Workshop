# 培訓室工具與材料管理系統

本網站用來查詢及管理培訓室中的工具、材料、存放位置、使用狀態與數量。

## 登入

1. 開啟網站首頁。 [FRC8725 Workshop](https://frc8725.github.io/FRC8725-Workshop)
2. 按下「使用 Google 登入」。
3. 選擇你的 Google 帳號。
4. 系統會自動確認帳號權限。

如果畫面顯示「授權申請已送出」，代表帳號正在等待管理員核准。管理員核准後，重新整理頁面即可使用。

## 平面圖總覽

登入後會進入培訓室平面圖總覽。

- 點擊平面圖上的櫃子或區域，可以查看該位置存放的物品。
- 左側搜尋區可以快速尋找工具與材料。
- 右側狀態摘要會顯示物品總數、工具、材料、低存量等資訊。

統計時，工具依總件數計算，使用中的工具依「總數量－可用數量」計算；材料則每筆資料算一種類。

## 搜尋與篩選

在搜尋欄輸入下列任一內容即可搜尋：

- 物品名稱
- 標籤
- 說明
- 櫃子名稱
- 抽屜或層板名稱

可以使用篩選按鈕只顯示：

- 全部
- 工具
- 材料
- 低存量或缺貨物品

搜尋結果可以切換為卡片或列表顯示。系統會記住你的顯示偏好。

## 查看櫃子內容

進入櫃子頁面後，可以：

- 查看各抽屜、層板或區域中的物品
- 在目前櫃子內搜尋
- 查看工具狀態
- 查看材料數量與低存量提示
- 切換卡片或列表顯示

按下「返回平面圖」即可回到總覽。

## 新增物品

1. 在平面圖總覽或櫃子頁面按下「新增物品」。
2. 選擇物品類型：工具或材料。
3. 填寫名稱、存放位置及其他資料。
4. 按下「新增物品」完成儲存。

新增工具時只需設定總數量，系統會自動將全部工具設為可用；編輯工具時可分別調整可用數量與總數量。材料可以設定數量、單位與最低存量。

## 修改物品

1. 找到要修改的物品。
2. 按下編輯按鈕。
3. 修改資料。
4. 按下「儲存變更」。

## 刪除物品

1. 找到要刪除的物品。
2. 按下刪除按鈕。
3. 確認物品名稱與位置。
4. 在確認視窗中按下「刪除」。

刪除後無法復原，請先確認資料是否正確。

## 工具與材料資訊

工具會顯示目前狀態與「可用數量／總數量」，例如 `1/3`：

- 可用
- 使用中
- 願望清單

材料會以數量緊接單位的格式顯示，例如 `1個`。當數量低於最低存量時，系統會顯示低存量或缺貨提示。

工具可透過「使用」與「歸還」按鈕快速調整數量；材料可透過「取用」與「補充」按鈕快速調整數量。每次操作會增減 1，數量不會低於 0。

## 標籤

新增或編輯物品時，可以輸入標籤並按 Enter 建立。

標籤可用來分類物品，也可以透過搜尋欄尋找具有特定標籤的物品。

## 登出

按下側邊欄帳號資訊下方的「登出」即可離開系統。

在共用電腦上使用完畢後，請務必登出。

## 常見問題

### 登入後仍顯示等待授權

請等待管理員核准帳號，核准後重新整理頁面。

### 找不到物品

請確認搜尋文字與篩選條件，或先切換到「全部」。

### 無法新增、修改或刪除資料

請確認網路連線與登入狀態。如果問題持續發生，請聯絡系統管理員。

### 畫面沒有顯示最新內容

按下瀏覽器重新整理，或使用 `Ctrl + Shift + R` 強制重新載入頁面。
